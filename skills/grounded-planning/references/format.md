Reference for user on how to use with Plannotator

The workflow

Step 1: Spawn planner subagent → it writes plan.md
Step 2: You run /plannotator-annotate plan.md → visual review in browser

| Section             | Combined Format                                                                |
| :------------------ | :----------------------------------------------------------------------------- |
| Context             | ✅ Why the change                                                              |
| Approach            | ✅ Recommended approach only                                                   |
| Steps               | ✅ - [ ] checklist items; ✅ Phased steps with evidence + confidence + details |
| Files to Modify     | ✅ Expl`icit list Ï                                                            |
| Reuse               | ✅ Existing code to leverage                                                   |
| Verification        | ✅ How to test end-to-end; ✅ Checkpoints per phase                            |
| Evidence Pack       | ✅ Claims with source, confidence, implication                                 |
| Uncertainty         | ✅ Low/Medium/High classification level                                        |
| Bottom Line         | ✅ Per-step confidence, key risk, gaps                                         |
| Decision Points     | ✅ "If X then Y" with evidence                                                 |
| Risks & Mitigations | ✅ Implicit; ✅ Explicit with evidence citations                               |
| Source              | ✅ Every claim cites a source traceability                                     |
